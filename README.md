# Pokemon Explorer

A single-page HTML app for looking up Pokemon by name using the [PokeAPI](https://pokeapi.co/).

## Features

- Search for any Pokemon by name
- Displays sprite, height, weight, and types
- Pokeball loading animation while fetching
- Dark mode toggle

## Usage

Just open [index.html](index.html) in a browser — no build step or server required.

1. Type a Pokemon name into the input field
2. Click **Search**
3. View the result card with the Pokemon's stats

## Tech

Plain HTML, CSS, and JavaScript. Data comes from `https://pokeapi.co/api/v2/pokemon/{name}`.
